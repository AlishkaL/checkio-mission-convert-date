Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę podzielić podany ciąg znaków na części. Zbuduj z nich obiekt daty i przekonwertuj ten obiekt daty z powrotem na ciąg w poprawnym formacie. Po każdym kroku proszę sprawdzić, czy aktualne dane są prawidłowe, jeśli nie, proszę natychmiast zwrócić wyrażenie.

Ok, od czego powinienem zacząć?

Proszę podzielić podany ciąg znaków na osobne dni, miesiące i lata za pomocą metody str.split() z separatorem "/". 

dd, mm, yyyy = date.split("/")

Proszę to zrobić wewnątrz try/except, aby sprawdzić, czy data jest prawidłowa i złapać wyjątek ValueError. 

try:
    dd, mm, yyyy = date.split("/")
except ValueError:
    return "Error: Invalid date."
Co robić dalej?

Najpierw proszę zaimportować moduł datetime i przeczytać o klasie date() i metodzie date.isoformat(). 

import datetime

Zbuduj obiekt daty z oddzielnych części, przekonwertowanych na liczbę całkowitą, za pomocą date(). Proszę to zrobić wewnątrz wyrażenia try/except i przechwycić wyjątek ValueError. 

try:
    D = datetime.date(int(yyyy), int(mm), int(dd))
except ValueError:
    return "Error: Invalid date."
Nadal potrzebuję podpowiedzi.

Proszę przekonwertować obiekt daty do formatu ISO za pomocą date.isoformat() i sprawdzić, czy jest równy ciągowi znaków, bezpośrednio zbudowanemu z oddzielnych części. 

res = D.isoformat()
if res!= f"{yyyy}-{mm}-{dd}":
    return "Error: Invalid date."
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi, zwrócić wynik i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Oto całe rozwiązanie:

import datetime


def convert_date(date: str) -> str:
    
    try:
        dd, mm, yyyy = date.split("/")
    except ValueError:
        return "Error: Invalid date."
    
    try:
        D = datetime.date(int(yyyy), int(mm), int(dd))
    except ValueError:
        return "Error: Invalid date."
    
    res = D.isoformat()
    if res!= f"{yyyy}-{mm}-{dd}":
        return "Error: Invalid date."
    
    return res
